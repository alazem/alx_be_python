### Python Introduction Tasks

This repository contains basic Python exercises to practice arithmetic operations, user input handling, and simple calculations.

#### Tasks Overview

1. **Basic Arithmetic Exercise**  
   - **File**: `basic_operations.py`  
   - Perform addition, subtraction, and multiplication of predefined numbers.

2. **Simple Interest Calculator**  
   - **File**: `simple_interest.py`  
   - Calculate interest using the formula \( I = P \times R \times T \).

3. **Calculate the Area of a Rectangle**  
   - **File**: `rectangle_area.py`  
   - Compute the area using predefined length and width.

4. **Convert Hours to Seconds**  
   - **File**: `hours_to_seconds.py`  
   - Convert a given number of hours into seconds.

5. **User Input Age Calculator**  
   - **File**: `future_age_calculator.py`  
   - Calculate the user’s age in 2050 based on their current age.

6. **Personal Finance Calculator** *(Advanced)*  
   - **File**: `finance_calculator.py`  
   - Compute monthly savings and project annual savings with interest.
